# Yulu Hypothesis Testing

## Project Overview
This project analyzes Yulu bike rental data using hypothesis testing and statistical analysis techniques in Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Google Colab

## Files Included
- Yulu_Bikes.ipynb
- Yulu_Bikes.pdf

## Analysis Performed
- Exploratory Data Analysis
- Hypothesis Testing
- Statistical Analysis
- Data Visualization

## Key Insights
- Demand pattern analysis
- Seasonal impact analysis
- Working day vs non-working day analysis
- Weather impact on rentals
